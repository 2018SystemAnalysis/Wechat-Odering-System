# API设计
本项目的API将按照功能为划分进行设计

## 与用户相关的接口
'POST /register' 新用户注册

'GET /login' 用户登录

'POST /bindPhone' 用户绑定手机号

'GET /getMyInfo' 获取用户信息

## 与饮品相关的接口
'GET /getfoodList' 获取饮品列表

## 与管理员相关的接口
'POST /admin/login' 管理员登录

'POST /admin/addFood' 后台添加菜品

'POST /admin/updateFood' 更新菜品

'GET /admin/deleteFood' 删除菜品
